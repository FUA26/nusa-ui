Nusa UI provides a collection of production-ready components that you own completely. Install them directly into your project, customize them to your needs, and ship with confidence.

## Why Nusa UI?

| Feature | Description |
| --- | --- |
| Built on Radix UI | Uses accessible primitives as the foundation |
| Accessible by default | ARIA-compliant patterns and keyboard support |
| CLI installation | Add components with the shadcn CLI |
| Full ownership | No runtime dependency, only source code |
| Tailwind CSS v4 | Utility-first styling with local control |
| TypeScript support | Strong typing across components and blocks |

## Philosophy

- Ownership over dependency
- Accessible first
- Minimal footprint
- Customization freedom
