```tsx

  return <Button variant="outline">Button</Button>
}
```

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"default" | "outline" | "secondary" | "ghost" | "destructive" | "link"` | `"default"` | Visual style of the button. |
| `size` | `"default" | "xs" | "sm" | "lg" | "icon" | "icon-xs" | "icon-sm" | "icon-lg"` | `"default"` | Button sizing preset. |
| `asChild` | `boolean` | `false` | Renders the button as a child element. |
