## API Reference

All standard HTML textarea attributes are supported.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `placeholder` | `string` | - | Placeholder text. |
| `disabled` | `boolean` | `false` | Disables the textarea. |
| `rows` | `number` | - | Number of visible text lines. |
